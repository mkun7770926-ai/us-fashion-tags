# 大鹅组-US服装带货标签

这是一个用于美国 TikTok Shop 服装带货的标签随机生成与一键复制工具。

## 文件说明

- `index.html`：网站主页面，GitHub Pages 必须使用这个文件名作为入口。
- `.nojekyll`：告诉 GitHub Pages 这是普通静态网页，不需要 Jekyll 处理。
- `README.md`：项目说明，不影响网页运行。

## 使用方式

打开网页后：

1. 选择“大促模式”或“常规模式”。
2. 点击对应商品类型。
3. 系统随机生成 5 个英文标签并自动复制。
4. 英文标签下方显示中文注释，但中文不会被复制。
5. 同一类型近期生成的结果会尽量避免连续重复。

## 更新网页

后续更新时，只需要用新的 `index.html` 替换仓库里的旧文件并提交即可。
